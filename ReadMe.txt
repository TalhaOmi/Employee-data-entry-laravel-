1. Install Laravel 8 in Xampp folder
2. Paste the "employee_management" folder in the "htdocs" folder inside xampp.
3. Create a database in phpmyadmin naming "talent_centric" and export the file from the database folder.
4. Run PHP artisan serve and it will give a http://127.0.0.1:8000/ link.
5. Good to go.
